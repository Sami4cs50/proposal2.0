Add your song here as: song.mp3

The page is already wired to play "music/song.mp3" when the
"Play Music" button is clicked (see the <audio> tag in index.html).

To use a different song later, either:
  1) Replace this file with your own MP3, keeping the exact name "song.mp3", or
  2) Open index.html and change the src="music/song.mp3" value on the
     <audio id="bgMusic" ...> line to point to your new file.

No other code changes are needed.
